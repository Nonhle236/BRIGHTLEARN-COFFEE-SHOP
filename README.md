# ☕ Bright Coffee Shop Sales Analysis

## 📌 Project Overview
This project analyzes the **Bright Coffee Shop Sales dataset** to uncover insights that can help improve business performance and increase revenue.  

As a Junior Data Analyst, the goal is to support decision-making by identifying key trends in product sales, customer behavior, and time-based performance.

---

## 🎯 Objectives
The analysis focuses on answering the following questions:

- Which products generate the most revenue?
- What time of day does the store perform best?
- What are the sales trends across products and time intervals?
- What recommendations can improve sales performance?

---

## 🗂️ Dataset Description
The dataset contains daily transactional data, including:

- Transaction Date  
- Transaction Time  
- Product Name  
- Quantity Sold  
- Unit Price  
- Store / Location (if applicable)
